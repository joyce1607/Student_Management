# Student Management System

A simple console-based Student Management System developed using Java. This project allows users to manage student records with features like adding, viewing, searching, and deleting students.

## Features

- Add Student Details
- View All Students
- Search Student by ID
- Delete Student Record
- Console-Based User Interface

## Technologies Used

- Java
- ArrayList
- Object-Oriented Programming (OOP)
- VS Code / Eclipse
